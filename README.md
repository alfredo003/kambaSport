
# Sistema de gestão de Ginásios
 <img src="./screen/logo.png" width="400"/><br>
kambaSport é um sistema de gestão de ginásio focado em ajudar na realização de algumas tarefas concernente a gestão dos clientes e controler dos funcionarios.
### Funcionalidades do kambaSport:
- Controle De Funcionários<br>
- Controle De Clientes<br>
-	Mensalidades<br>
-	controle financeiro<br>	 
### Ferramentas utilizadas no desenvolvimento :
Back-end: <b>Linguagem PHP</b>
Front-end: <b>jQuery, Bootstrap</b> 
Base de Dados: <b>Mysql</b>
